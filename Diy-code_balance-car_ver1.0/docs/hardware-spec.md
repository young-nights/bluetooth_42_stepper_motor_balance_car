## 7. PID 参数

| 控制环 | 参数 | 当前值 | 说明 |
|--------|------|--------|------|
| **直立环** | Kp | 15.0 | 比例增益 |
| | Ki | 0 | 不使用积分项 |
| | Kd | 4.0 | 微分阻尼（抑制高频振荡） |
| | 输出限幅 | ±1300 | 占空比限幅 |
| **速度环** | Kp | 0（已禁用） | 加速度计积分估算速度噪声过大 |
| | Ki | 0（已禁用） | 暂未启用，待编码器方案 |

> **注意**：速度环当前已禁用（Kp=0, Ki=0），因为速度估算基于 MPU6050 加速度计积分，
> 在无编码器的步进电机系统中，积分漂移噪声过大，会干扰直立环调试。
> 建议后续加装编码器后再重新启用速度环。
